<h2>安装</h2>



#### npm安装

通过一下命令安装vue-brisk。

```shell
npm i vue-brisk -S
```



#### CDN引入

如果为了减小打包体积，可以使用CDN引入。

```html
<!-- 引入样式 -->
<link rel="stylesheet" href="https://unpkg.com/vue-brisk/lib/vue-brisk.css">
<!-- 引入组件库 -->
<script src="https://unpkg.com/vue-brisk/lib/vue-brisk.umd.min.js"></script>
```
